# Trichella

Public repo: **https://github.com/darderrdur17/Trichella**

## Deploy to Vercel

1. Go to [vercel.com](https://vercel.com) → **Add New** → **Project**.
2. **Import** `darderrdur17/Trichella` (or connect your GitHub and select this repo).
3. Set **Root Directory** to `trichella-app` (Edit → Root Directory → `trichella-app`).
4. Leave **Framework Preset** as Vite (or let Vercel detect it).
5. Add any **Environment Variables** your app needs (e.g. API keys) in Project Settings.
6. Click **Deploy**.

The app uses `trichella-app/vercel.json` for build command, output directory (`dist`), and API route config.
